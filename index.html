Guide-Compress

Hay distintas maneras de comprimir o descomprimir directorios. A continuación veremos cómo se puede hacer con archivos tar.gz, gz, bz2, zip y rar

Directorios tar.gz

Comprimir:

$ tar -zcvf nombre-archivo.tar.gz nombre-directorio

Donde:

  • -z: Comprimir archivos usando gzip
  • -c: Crear un nuevo archivo
  • -v: Mostrar el proceso durante la creación del archivo
  • -f: nombre de archivo

Descomprimir:

$ tar -zxvf nombre-archivo.tar.gz nombre-directorio

Donde:

  • -z: Comprimir archivos usando gzip
  • -x: Extrae el contenido del archivo comprimido
  • -v: Mostrar el proceso durante la creación del archivo
  • -f: nombre de archivo

Directorios gz

Comprimir:

$ gzip -9 directorio

Descomprimir:

$ gzip -d directorio.gz

Directorios bz2

Comprimir:

$ bzip fichero

Descomprimir:

$ bzip2 -d fichero.bz2

Ni los directorios gz ni bz2 pueden comprimir directorios, sólo archivos. En caso de desear empaquetar directorios enteros se debe usar tar.gz

Directorios zip

Comprimir:

$ zip archivo.zip directorio

Descomprimir:

$ unzip archivo.zip

Directorios rar

Comprimir:

$ rar -a archivo.rar directorio

Donde:

  • -a: selecciona todo lo que haya en el directorio para comprimirlo en el archivo.rar

Descomprimir:

$ rar -x archivo.rar

Donde:

  • -x: extrae los elementos de archivo.rar
